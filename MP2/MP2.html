MP2 Write-Up

Overview

This project involved creating a game using P5 and P5.play libraries. The game consists of a bumble bee trying to collect all of the flowers as if it is collecting the pollen from them. The user uses their trackpad to move their mouse which controls the bumble bee. When they move their mouse over a flower it disappears and the user gains a point. If the user gets all 20 flowers before they fall past the bottom of the screen, the user wins. Otherwise the user will recieve their score and can try again by reloading the page.

Development Process

Creating this game I began using p5 online to design the base of the game which allowed me to remove falling blocks when the user dragged their mouse, along with a pink box, onto each of the objects. From here I took the code into visual studio and loaded the images to use as sprites rather than blocks. I created the flower image in adobe and found the bee image online using stock free images. Once the images were preloaded I created a sprite that was the bee and a group for the flower sprites. I then scaled the images of the bee and the flowers to better fit the game. Next I added end game text. When the flower group was completely removed by the player a winning message appears and if a user is unable to collect all 20 flowers, a losing message will appear as well as the users score. Every time a user removes a flower, the score number on the screen and the score variable increase by one. To restart the game the user must reload their browser screen.
