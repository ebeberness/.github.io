FP Proposal

Description of Project

I will create a drawing board using p5 that will allow users to create art using different tools and download the canvas onto their device.

Drawings of your envisioned interface

Project Version Descriptions

Prototype version: The MVP will allow users to draw basic things with their cursor, a paintbrush, in black and white. It will allow the user to change the background to different colors. The canvas will also download the drawing to the users device.

Final Version - Your envisioned final submission: The final version will include allow the user to use their cursor, represented oringinally on the screen as a paintbrush, to draw on the specified canvas area of the page. They will be able to choose the color they are drawing and draw with circles or squares if the user changes their cursor using buttons on the side of the screen. There will be a button to download the art on the screen as well.

    5. Figure out the code for saving the canvas and add to p5.
    6. Research how to turn cursor into a pen or paintbrush image and insert the code into p5.
